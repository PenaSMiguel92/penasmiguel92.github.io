EMG Controlled Robotic Hand

Demonstrates ability to design, implement, and complete a complex project.

A Work in Progress

Capstone project consisting of several functional parts. The robot hand built with Autodesk Fusion 360 CAD Software, printed with Creality Ender 3, and assembled with servo motors. EMG signal acquisition hardware, circuit designed with EasyEDA software (an Electronics Design Automation tool), assembled on breadboard, and signals acquired by Arduino Uno. Software developed with Arduino IDE ,C#, Visual Studio 2019, .NET Framework, and WPF. The electrodes are simply bare wires, petroleum jelly, and tape.

Technology Utilized

  • Autodesk Fusion 360 CAD Software
  • Creality Ender 3 3D Printer with PLA Filament
  • EasyEDA software for circuit design
  • Arduino UNO programmed with Arduino IDE
  • C#, Visual Studio 2019, .NET Framework, and WPF
